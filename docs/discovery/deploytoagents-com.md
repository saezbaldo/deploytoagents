Audit an application's current readiness for agent discovery.

Retrieve persisted audit scores, findings, and evidence URLs.

Generate a distribution plan from observed technical gaps.

Add official-registry, public-repository, customer-evidence, and independent-discovery actions to that plan.

Invoke these capabilities through the public remote MCP server.
